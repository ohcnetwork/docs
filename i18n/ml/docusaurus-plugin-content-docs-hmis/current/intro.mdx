# HMIS പ്ലേബുക്കുകൾ

ആശുപത്രി മാനേജ്മെന്റ് പ്ലേബുക്കുകൾ **ഫ്ലോകൾ** യഥാർത്ഥ ഫ്രണ്ട്-ഡെസ്ക്, വാർഡ് സാഹചര്യങ്ങളുമായി ബന്ധിപ്പിക്കുന്നു. ഓരോ പ്ലേബുക്കും ഉൾപ്പെട്ട ഫ്ലോകൾ, പ്രതീക്ഷിക്കുന്ന ഓപ്പറേറ്റർ പാത, HMIS ഡിപ്ലോയ്മെന്റുകൾക്ക് നിർദ്ദിഷ്ട എഡ്ജ് കേസുകൾ പട്ടികപ്പെടുത്തുന്നു.

## പ്ലേബുക്കുകൾ കോർ ഡോക്സുമായി എങ്ങനെ ബന്ധപ്പെടുന്നു

| പാളി | സ്ഥാനം | ഉദാഹരണം |
| --- | --- | --- |
| ആശയം | [കോർ ഡോക്സ്](/intro) | രോഗി എന്താണ്? |
| ഫ്ലോ | [കോർ ഡോക്സ്](/flows/clinical/create-patient) | രോഗിയെ എങ്ങനെ സൃഷ്ടിക്കാം? |
| പ്ലേബുക്ക് | ഇവിടെ | ഡ്യൂപ്ലിക്കേറ്റുകളും വാക്ക്-ഇനുകളും ഉള്ള ഔട്ട്‌പേഷ്യന്റ് രജിസ്ട്രേഷൻ |

ഈ ഘടന പുതിയതാണെങ്കിൽ [ഡോക്യുമെന്റേഷൻ ഹോം](/intro) മുതൽ ആരംഭിക്കുക.
