# ഔട്ട്‌പേഷ്യന്റ് രജിസ്ട്രേഷൻ

ആശുപത്രി ഫ്രണ്ട് ഡെസ്കിൽ വാക്ക്-ഇൻ അല്ലെങ്കിൽ ഷെഡ്യൂൾ ചെയ്ത ഔട്ട്‌പേഷ്യന്റിനെ രജിസ്റ്റർ ചെയ്യുക. ഈ പ്ലേബുക്ക് [രോഗിയെ സൃഷ്ടിക്കുക](/flows/clinical/create-patient) ഫ്ലോയും പ്രൊഡക്ഷനിൽ രജിസ്ട്രേഷൻ ടീം നേരിടുന്ന HMIS-നിർദ്ദിഷ്ട എഡ്ജ് കേസുകളും സംയോജിപ്പിക്കുന്നു.

## ഉൾപ്പെട്ട ഫ്ലോകൾ

| ക്രമം | ഫ്ലോ | ഉദ്ദേശ്യം |
| --- | --- | --- |
| 1 | [രോഗിയെ സൃഷ്ടിക്കുക](/flows/clinical/create-patient) | നിലവിലില്ലാത്ത പുതിയ സന്ദർശകൻ |
| 2 | രോഗി തിരയൽ & മർജ് *(ഉടൻ)* | സൃഷ്ടിക്കുന്നതിന് മുമ്പ് സംശയാസ്പദ ഡ്യൂപ്ലിക്കേറ്റ് |
| 3 | OP എൻകൗണ്ടർ ആരംഭിക്കുക *(ഉടൻ)* | രജിസ്ട്രേഷന് ശേഷം ഇന്നത്തെ സന്ദർശനം തുറക്കുക |

## സാധാരണ പാത

1. റിസപ്ഷൻ ഫോൺ അല്ലെങ്കിൽ ദേശീയ ID കൊണ്ട് തിരയുക.
2. പൊരുത്തമില്ല → **പുതിയ രോഗി രജിസ്റ്റർ** → സൗകര്യ ചോദ്യാവലി പ്രകാരം ഡെമോഗ്രാഫിക്സ് പൂർത്തിയാക്കുക.
3. പ്രവർത്തനക്ഷമമാക്കിയാൽ OPD സ്ലിപ്പ് / ടോക്കൺ പ്രിന്റ് ചെയ്യുക.
4. ട്രയേജ് അല്ലെങ്കിൽ കൺസൾട്ടിംഗ് റൂം ക്യൂവിലേക്ക് രോഗിയെ അയയ്ക്കുക.

അധിക കോൺഫിഗറേഷൻ ഇല്ലാതെ ഇത് കോർ [രോഗിയെ സൃഷ്ടിക്കുക](/flows/clinical/create-patient) ഫ്ലോയുമായി പൊരുത്തപ്പെടുന്നു.

## എഡ്ജ് കേസുകൾ

### സംശയാസ്പദ ഡ്യൂപ്ലിക്കേറ്റ്

**ലക്ഷണം:** പേര്, DOB, ഫോൺ നൽകിയതിന് ശേഷം ഡ്യൂപ്ലിക്കേറ്റ് മുന്നറിയിപ്പ്.

**നടപടി:**

1. നിർദ്ദേശിച്ച പൊരുത്തം സൈഡ്-ബൈ-സൈഡ് തുറക്കുക.
2. ഒരേ വ്യക്തിയാണെങ്കിൽ → **നിലവിലുള്ള രേഖ ഉപയോഗിക്കുക**; രണ്ടാമത്തെ രോഗി സൃഷ്ടിക്കരുത്.
3. ഒരേ പേരുള്ള വ്യത്യസ്ത വ്യക്തിയാണെങ്കിൽ → ഓവർറൈഡ് കാരണം രേഖപ്പെടുത്തി (സൗകര്യം ആവശ്യപ്പെടുമ്പോൾ) തുടരുക.
4. ഉറപ്പില്ലെങ്കിൽ → രജിസ്ട്രേഷൻ സൂപ്പർവൈസറിലേക്ക് എസ്കലേറ്റ്; HMIS ഓഡിറ്റ് ലോഗ് തീരുമാനം സൂക്ഷിക്കുന്നു.

### അജ്ഞാത ഐഡന്റിറ്റി (എമർജൻസി)

**ലക്ഷണം:** രോഗിക്ക് പേരോ ID-യോ നൽകാൻ കഴിയില്ല (ട്രോമ, അബോധാവസ്ഥ).

**നടപടി:**

1. സൗകര്യത്തിന്റെ **അജ്ഞാത രോഗി** നാമകരണം ഉപയോഗിച്ച് താൽക്കാലിക രേഖ സൃഷ്ടിക്കുക (ഉദാ. `Unknown Male ED-042`).
2. എമർജൻസി MRN നൽകുക; സാധ്യമെങ്കിൽ തിരിച്ചറിയൽ അടയാളങ്ങൾ ഫോട്ടോ എടുക്കുക.
3. തിരിച്ചറിഞ്ഞതിന് ശേഷം പൂർണ്ണ ഡെമോഗ്രാഫിക് രേഖയിലേക്ക് മർജ് *(മർജ് ഫ്ലോ ഉടൻ)*.

മിക്ക HMIS ഡിപ്ലോയ്മെന്റുകളിലും `emergency_registration` അനുമതി ആവശ്യമാണ്.
