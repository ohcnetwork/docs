# പാലിയേറ്റീവ് പ്ലേബുക്കുകൾ

പാലിയേറ്റീവ് കെയർ പ്ലേബുക്കുകൾ **ഫ്ലോകൾ** വീട്ടിലും സൗകര്യ പാലിയേറ്റീവ് പ്രോഗ്രാമുകളുമായി ബന്ധിപ്പിക്കുന്നു. ഓരോ പ്ലേബുക്കും ഉൾപ്പെട്ട ഫ്ലോകൾ, കെയർ ടീം പാത, പാലിയേറ്റീവ് ഡിപ്ലോയ്മെന്റുകൾക്ക് നിർദ്ദിഷ്ട എഡ്ജ് കേസുകൾ പട്ടികപ്പെടുത്തുന്നു.

## പ്ലേബുക്കുകൾ കോർ ഡോക്സുമായി എങ്ങനെ ബന്ധപ്പെടുന്നു

| പാളി | സ്ഥാനം | ഉദാഹരണം |
| --- | --- | --- |
| ആശയം | [കോർ ഡോക്സ്](/intro) | രോഗി എന്താണ്? |
| ഫ്ലോ | [കോർ ഡോക്സ്](/flows/clinical/create-patient) | രോഗിയെ എങ്ങനെ സൃഷ്ടിക്കാം? |
| പ്ലേബുക്ക് | ഇവിടെ | കമ്മ്യൂണിറ്റി പാലിയേറ്റീവ് കെയറിൽ രോഗിയെ എൻറോൾ ചെയ്യൽ |

ഈ ഘടന പുതിയതാണെങ്കിൽ [ഡോക്യുമെന്റേഷൻ ഹോം](/intro) മുതൽ ആരംഭിക്കുക.
