# പാലിയേറ്റീവ് രോഗി എൻറോൾമെന്റ്

കമ്മ്യൂണിറ്റി അല്ലെങ്കിൽ സൗകര്യ-അടിസ്ഥാന പാലിയേറ്റീവ് പ്രോഗ്രാമിൽ രോഗിയെ എൻറോൾ ചെയ്യുക. ഈ പ്ലേബുക്ക് [രോഗിയെ സൃഷ്ടിക്കുക](/flows/clinical/create-patient) ഫ്ലോയും പാലിയേറ്റീവ്-നിർദ്ദിഷ്ട ഇൻടേക്ക്, കെയർഗിവർ ക്യാപ്ചർ, പ്രോഗ്രാം അസൈൻമെന്റും സംയോജിപ്പിക്കുന്നു.

## ഉൾപ്പെട്ട ഫ്ലോകൾ

| ക്രമം | ഫ്ലോ | ഉദ്ദേശ്യം |
| --- | --- | --- |
| 1 | [രോഗിയെ സൃഷ്ടിക്കുക](/flows/clinical/create-patient) | നിലവിലില്ലാത്ത പുതിയ രോഗി |
| 2 | രോഗി തിരയൽ | ഡ്യൂപ്ലിക്കേറ്റുകൾ സൃഷ്ടിക്കുന്നതിന് മുമ്പ് തിരിച്ചുവരുന്ന രോഗികളെ പൊരുത്തിപ്പിക്കുക |
| 3 | പാലിയേറ്റീവ് പ്രോഗ്രാം എൻറോൾമെന്റ് *(ഉടൻ)* | പ്രോഗ്രാം, കെയർ ടീം, സന്ദർശന ഇടവേള നൽകുക |

## സാധാരണ പാത

1. കമ്മ്യൂണിറ്റി നഴ്സ് അല്ലെങ്കിൽ ഇൻടേക്ക് കോർഡിനേറ്റർ ഫോൺ, ദേശീയ ID, പേര് കൊണ്ട് തിരയുന്നു.
2. പൊരുത്തമില്ല → **പുതിയ രോഗി രജിസ്റ്റർ** → ഡെമോഗ്രാഫിക്സും പ്രാഥമിക രോഗനിർണയവും പൂർത്തിയാക്കുക.
3. പ്രാഥമിക കെയർഗിവറിന്റെ പേര്, ഫോൺ, ബന്ധം രേഖപ്പെടുത്തുക.
4. പാലിയേറ്റീവ് പ്രോഗ്രാം അസൈൻ ചെയ്യുക (വീട്ടിൽ, സൗകര്യം, അല്ലെങ്കിൽ ഹൈബ്രിഡ്).
5. ആദ്യ അസസ്മെന്റ് സന്ദർശനം ഷെഡ്യൂൾ ചെയ്ത് പാലിയേറ്റീവ് കെയറിനുള്ള സമ്മതം രേഖപ്പെടുത്തുക.

## എഡ്ജ് കേസുകൾ

### ആശുപത്രി IP-യിൽ നിന്ന് റഫർ ചെയ്തു

**ലക്ഷണം:** ഡിസ്ചാർജ് സമ്മറിയിൽ രോഗി; കുടുംബം വീട്ടിലെ പാലിയേറ്റീവ് ഫോളോ-അപ്പ് അഭ്യർത്ഥിക്കുന്നു.

**നടപടി:**

1. ആശുപത്രി MRN അല്ലെങ്കിൽ റഫറൽ സ്ലിപ്പിലെ ഫോൺ കൊണ്ട് തിരയുക.
2. രോഗി നിലവിലുണ്ടെങ്കിൽ → രേഖ തുറന്ന് പാലിയേറ്റീവ് പ്രോഗ്രാം എൻറോൾമെന്റ് ചേർക്കുക; ഡ്യൂപ്ലിക്കേറ്റ് സൃഷ്ടിക്കരുത്.
3. ഡിസ്ചാർജ് സമ്മറിയും സജീവ മരുന്ന് ലിസ്റ്റും ഇംപോർട്ട്/അറ്റാച്ച് ചെയ്യുക.
4. നിയോഗിച്ച കമ്മ്യൂണിറ്റി നഴ്സിനായി **കെയർ മാറ്റം** ഫ്ലാഗ് ചെയ്യുക.
